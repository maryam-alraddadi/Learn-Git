# الامر git add


لنقل الملف من مرحلة untracked إلى مرحلة staged نستخدم الامر

</div>

```
git add fileName
```

<div dir="rtl">
لاضافة اكثر من ملف إلى مرحلة الstaged نستخدم الامر

</div>

```
git add .
```
<div dir="rtl">
هذا الامر يضيف كل الملفات الجديدة أو المعدلة في المجلد الحالي إلى مرحلة الstaged




</div>
