# الامر git commit


بعد نقل الملفات إلى مرحلة الstaged باستخدام 

</div>

```
git add
```

<div dir="rtl">
تكون جاهزة لعمل commit واضافتها للrepository

لعمل commit نحتاج الى تحديد رسالة توضح التغييرات في هذه الcommit
</div>

```
git commit -m "write message here"
```
