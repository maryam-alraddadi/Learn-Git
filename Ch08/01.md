# إنشاء وعرض tag


في git نستطيع عمل tag على أي نقطة في المشروع 

وأحد استخدامات الtags هو تسمية الاصدارات



لعرض الtags الخاصة بالمشروع 


</div>

```
git tag
```


<div dir="rtl">

لانشاء تاق جديدة عند النقطة الحالية للمشروع باسم v1.0.0


</div>


```
git tag v1.0.0
```
